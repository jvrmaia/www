# João Maia - Personal Website

A static website showcasing my professional profile, built from my LinkedIn export.

## Features

- Clean, modern design
- Fully responsive (mobile, tablet, desktop)
- Professional layout highlighting experience, skills, and certifications
- Fast loading static HTML/CSS
- No dependencies required

## Structure

- `index.html` - Main HTML file
- `styles.css` - All styling and responsive design
- `README.md` - This file

## Viewing the Website

Simply open `index.html` in any modern web browser. No server or build process required.
